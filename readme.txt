== Description ==

= Plugin Features =


* Easy to use.
* Fully Responsive.
* Full customize.
* Custom Background Color.
* Custom Preloader Image.
* Retina Ready.
* jQuery & CSS3.
* Compatible with Google Chrome, FireFox, Opera, Safari, IE9, IE10, IE11 and with mobile devices like iPad, iPhone, Android, etc.


== Installation ==

= Installation =

1. Upload 'tcbd-preloader' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
